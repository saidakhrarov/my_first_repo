#Podskazkı po komandnoy stroke

Komanda smenı direktorii
```sh
cd c:\folder_name
```

Komanda otobrajenıya tekushıy direktorii
Macos Linux
```sh
pwd
```

Listing tekushiy direktoriii
windows:
```sh
dir
```
Linux Macos:
```sh
ls
```

Udalenıye fayla Windows:
```sh
del <filename>
```
v Linux, Macos ;
```sh
rm <filename>
```
